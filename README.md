# Flight Delay Analysis using Apache Spark (PySpark)

In this project, we analyze flight delay data using Apache Spark (PySpark). The goal is to understand how delays are affected by departure time and different flight routes.

## Overview

The analysis is based on real flight data and focuses on identifying patterns in delays. Using Spark, we process the dataset efficiently and explore how delays vary throughout the day.

## Technologies

* Python
* Apache Spark (PySpark)
* Google Colab
* Matplotlib

## Results

The results show that delays tend to increase during certain hours of the day, especially in the evening. Additionally, some routes consistently appear among those with higher average delays.

## Files

* flight_delay_analysis_spark.ipynb – main notebook with the analysis
* flights_2000.csv – dataset used
* avg_delay_by_hour.png – average delay per hour
* top10_routes_delay.png – routes with highest delays
* report.pdf – full report

## How to run

Open the notebook in Google Colab, upload the dataset, and run all cells to reproduce the results.
